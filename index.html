Как установить

Простые шаги для быстрой установки BlissDLC

1

Установите Java JDK 17

Убедитесь, что у вас установлена Java JDK версии 17 или выше

Скачать Java JDK 17
2

Скачайте клиент BlissDLC

Нажмите кнопку "Скачать сейчас" на этой странице

3

Переместите файл клиента

Поместите скачанный файл client.jar в удобную папку на диске C:

Пример: C:\BlissDLC\client.jar

4

Откройте командную строку

Откройте CMD (Командная строка) от имени администратора

5

Перейдите в папку с файлом

Введите команду для перехода в папку, где находится файл:

cd /d C:/ваш/путь/кфайлу
6

Запустите клиент

Введите команду для запуска BlissDLC:

java -jar client.jar

Готово! BlissDLC запустится

Важно: Файл должен находиться только на диске C:. Убедитесь, что Java JDK установлена правильно перед запуском.
